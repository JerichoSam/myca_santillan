Overview

Introduction

This project was built while following a tutorial from Alex The Analyst, where I learned to use Microsoft Power BI for data cleaning and visualization using Power Query Editor.

The dataset contains survey responses from data professionals covering job titles, salary ranges, favorite programming languages, and satisfaction levels with salary and work-life balance. The goal is to transform this raw survey data into clear, actionable insights.

3. Clean Data

Remove Unnecessary Columns

Columns not useful for analysis were identified and removed to reduce noise in the dataset.

Dropping Irrelevant Columns

Standardize Q1 — Job Title Column

  1. 1 Select the Q1 (Job Title) column
  2. 2 Click Split Column → By Delimiter
  3. 3 Use ( as the delimiter — Left-most occurrence
  4. 4 This isolates the clean job title from extra text in parentheses
  5. 5 Delete the second generated column
Q1 Column Selected

Result After Split & Column Removal

Standardize Q5 & Other Text Columns

  1. 1 Select the column to clean
  2. 2 Click Split Column → By Delimiter
  3. 3 Use colon : as delimiter at left-most occurrence
  4. 4 Remove the extra generated column
  5. 5 Repeat the process for all remaining text columns
Q5 Column Before Split
All Columns After Standardization

Standardize Salary Column

  1. 1 Duplicate the salary column to preserve original data
  2. 2 Use Split Column → By Digit to Non-Digit
  3. 3 Remove the column containing "k" suffix values
  4. 4 Use Replace Values to strip - and k
  5. 5 Convert to numeric data type

Create Average Salary Column

  1. 1 Click Add Column → Custom Column
  2. 2 Name it Average Salary
  3. 3 Formula: ([SalaryMin] + [SalaryMax]) / 2

Insights

Recommendations

💰 Improve Salary Satisfaction

With an average salary satisfaction of just 3.71/10, companies should review compensation structures and offer competitive benefits to retain data talent.

🐍 Focus on High-Demand Skills

Python dominates as the most popular language among data professionals by a wide margin — it remains the most valuable skill for aspiring analysts and scientists.

🚪 Create More Entry Opportunities

Nearly 32% of respondents found entering the data field difficult or very difficult. More training programs, internships, and entry-level pipelines are needed.
